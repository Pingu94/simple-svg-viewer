# Simple SVG Viewer

This is a very minimal SVG files viewer made with Electron.

![Svg Viewer example](img/simpleSvgViewer.png)

## To Use

- npm install
- npm start
- Drag and drop an svg file inside the application

![Svg Viewer example](img/drop_example.gif)

## To run tests

- npm test
